T-embed Flasher

Прошивальщик для ESP устройств с возможностью прошивки одним нажатием, без загрузки программ и ввода команд.

Разработчик: @m1ner203

T-Embed CC1101

Назад к выбору
Чип: ESP32-S3
Память: 8MB Flash
Модуль: CC1101 (433/868 MHz)
Интерфейс: USB-C

Доступные прошивки

Инструкция по прошивке

  1. Подключите T-Embed CC1101 к компьютеру через USB-C кабель
  2. Выберите нужную прошивку из списка выше
  3. Нажмите кнопку "Прошить" и следуйте инструкциям браузера
  4. Не отключайте устройство во время процесса прошивки

Использует ESP Web Tools для прямой прошивки через браузер

Требуется Chrome/Edge 89+ или Opera 76+ с поддержкой Web Serial API
